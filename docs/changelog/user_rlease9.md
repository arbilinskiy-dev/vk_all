## 🆕 Новые возможности

### Метки (ярлыки) для диалогов

**Что добавили:**  
В модуле «Сообщения» появились метки — цветные ярлыки, которые можно создавать и назначать на диалоги. Например: «Клиент», «VIP», «Спам», «Оплатил». Метки привязаны к проекту и видны только участникам проекта.

**Как использовать:**
- В списке диалогов над первым диалогом появилась панель меток
- Нажмите «+ Метка» — введите название и выберите цвет (5 цветов)
- Нажмите на метку — список диалогов отфильтруется по ней
- Откройте диалог — в шапке чата нажмите кнопку-тег 🏷 и назначьте нужные метки через чекбоксы
- Назначенные метки отображаются прямо в шапке чата как цветные бейджи
- В списке диалогов рядом с именем видны цветные точки назначенных меток
- Правый клик на метке — изменить название, цвет или удалить

**Что теперь можно:**
- Быстро классифицировать диалоги по категориям
- Фильтровать список диалогов одним кликом по нужной метке
- Визуально различать важные диалоги по цветным точкам в списке
- Видеть назначенные метки прямо в шапке чата

### Плавное раскрытие/сворачивание текста шаблона

**Что добавили:**  
Длинный текст шаблона теперь можно развернуть/свернуть прямо в списке одним кликом по кнопке «Показать полностью...» с плавной анимацией.

**Как использовать:**  
- В списке шаблонов длинный текст обрезается до 2 строк
- Нажмите «Показать полностью...» — текст плавно развернётся
- Повторное нажатие — плавно свернётся обратно

**Что теперь можно:**  
Просматривать полное содержимое шаблона без открытия редактора.

---

### Блокировка кнопок шаблона при пустых промокодах

**Что добавили:**  
Если шаблон использует промокоды, а в списке закончились свободные коды — кнопки «Вставить» и «Заменить» автоматически блокируются. Над кнопками появляется красный баннер с названием пустого списка.

**Как использовать:**  
- Если кнопки шаблона неактивны — проверьте красный баннер: он укажет какой список промокодов пуст
- Пополните список промокодов — кнопки автоматически разблокируются

**Что теперь можно:**  
Исключить случайную отправку сообщения без промокода.

## ✅ Решённые задачи

### Исправлен UI меток: обводки, размеры, палитра

**Проблема:**  
Элементы меток (чипы, кнопки, палитра цветов) были слишком мелкими — сложно читать и кликать. Палитра цветов закрывалась при попытке выбрать цвет. Иконки в сайдбаре диалогов были мельче, чем в сайдбаре проектов.

**Решение:**  
Увеличены размеры всех элементов меток (текст, отступы, иконки). Палитра уменьшена до 5 наиболее различимых цветов. Добавлены тонкие обводки к чипам для лучшей читаемости. Исправлен баг закрытия палитры. Иконки фильтров приведены к единому стилю с проектами.

---

### Улучшен интерфейс вкладки «Вложения» в диалогах

**Проблема:**  
Вкладка «Вложения» (фото, видео, ссылки, файлы в переписке) работала корректно, но элементы появлялись резко без плавных анимаций, числа не склонялись (например «из 1 сообщений»), а модальное окно просмотра фото не закрывалось по Escape.

**Решение:**  
Все списки и сетка фотографий теперь появляются плавно с каскадной анимацией. Переключение между категориями (Фото/Видео/Ссылки/Файлы) анимировано. Числа правильно склоняются: «из 1 загруженного сообщения», «Всего 5 вложений». Превью видео и ссылок показывают скелетон во время загрузки. Модалка просмотра фото закрывается по Escape и показывает курсор «уменьшить».
