# Управление Подборками Товаров (Альбомами)

**Ключевой файл:** `backend_python/services/market_album_service.py`
**Роутер:** `backend_python/routers/market.py`

Этот модуль отвечает за CRUD-операции с подборками товаров (в терминологии VK API — "альбомами").

---

## 1. Концепция

Подборки позволяют группировать товары в магазине сообщества VK. Бэкенд кеширует список подборок в таблице `market_albums`. Сервис позволяет создавать, переименовывать и удалять подборки, синхронизируя изменения с VK и локальным кешем.

---

## 2. API и Поток данных

### `POST /api/market/createAlbum`
-   **Payload:** `CreateAlbumPayload` (`projectId`, `title`).
-   **Сервис:** `create_market_album(db, project_id, title, user_token)`.
-   **Логика:** `market.addAlbum` → создание `MarketAlbum` в кеше → возврат схемы.

### `POST /api/market/editAlbum` (release2)
-   **Payload:** `EditMarketAlbumPayload` (`projectId`, `albumId`, `title`).
-   **Сервис:** `edit_market_album(db, project_id, album_id, title, user_token)`.
-   **Логика:** `market.editAlbum` → обновление `title` в кеше → возврат схемы.

### `POST /api/market/deleteAlbum` (release2)
-   **Payload:** `DeleteMarketAlbumPayload` (`projectId`, `albumId`).
-   **Сервис:** `delete_market_album(db, project_id, album_id, user_token)`.
-   **Логика:** `market.deleteAlbum` → удаление из кеша.

---

## 3. Оптимизации (release2)

### Приоритет админских токенов
Все три функции переведены на `call_vk_api_for_group()` (вместо `call_vk_api()`), которая приоритизирует токены администраторов указанной группы. Устранены лишние VK API Error 15 (Access denied).

### Защита от двойного клика
Хук `useProductsManager.ts` содержит флаг `isSavingAlbum`. Функции `handleCreateAlbum` и `handleSaveEditAlbum` проверяют флаг на входе, блокируют UI (`disabled={isSavingAlbum}`).

### Информационная подсказка
Иконка ℹ️ рядом с кнопкой «Все» в фильтрах. Тултип: подборки из ВКонтакте + ссылка на VK-раздел товаров.

### Пустые подборки (release2)
VK API `market.getAlbums` не возвращает альбомы с 0 товаров. В `refresh_all_market_data()` после синхронизации сохраняются локальные альбомы, отсутствующие в ответе VK, с `count = 0`.

### Ошибка 1402 (release2)
Код 1402 (Album not found) добавлен в `PERMANENT_ERROR_CODES` в `api_client.py` — без ретраев.
