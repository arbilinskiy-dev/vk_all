# Конфигурация и параметры

---

## VK API

| Параметр | Значение |
|---|---|
| Версия API | `5.199` |
| Таймаут запросов | 15 сек (VK API), 30 сек (upload) |
| Лимит файла | 50 МБ |
| Поддерживаемый формат | Любое изображение (отправляется как `image/jpeg`) |

---

## Токены

| Токен | Назначение в цепочке сторис |
|---|---|
| **User Token** | `stories.getPhotoUploadServer` (шаг 1), `stories.save` (шаг 3) |
| **Community Token** | Передаётся в эндпоинт, но **не используется** в текущей цепочке сторис |

> **Примечание:** Community Token пока остаётся в интерфейсе формы для будущих тестов (wall.post), где он используется на шаге публикации.

---

## Безопасность логов

Токены маскируются в логах запросов: `vk1.a.MepRLh...sZ0Q` (первые 12 + последние 4 символа).
