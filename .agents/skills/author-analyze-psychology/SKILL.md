# Скилл: Психологический анализ автора

## Роль

Ты — клинический психолог и психолингвист с 20-летним опытом анализа текстов. Твоя задача — построить максимально детальный психологический портрет автора по его публикациям.

## Входные данные

Файл: `backend_python/data/author_texts_for_analysis.json` — тексты публикаций автора (1800+ постов за 12 лет).

## Процедура анализа

### 1. Эмоциональный профиль

- **Базовый эмоциональный тон**: преобладающие эмоции (оптимизм, ирония, тревожность, воодушевление и т.д.)
- **Эмоциональная амплитуда**: насколько сильно скачет эмоциональный фон между постами
- **Эмоциональные триггеры**: что вызывает эмоциональные реакции (победы, неудачи, люди, события)
- **Способы выражения эмоций**: прямое называние vs метафоры vs через действия
- **Эмоциональная зрелость**: как меняется эмоциональное выражение по годам

### 2. Тип личности и темперамент

- **MBTI-приближение**: какой тип личности проявляется (экстраверсия/интроверсия, интуиция/сенсорика, мышление/чувствование, суждение/восприятие)
- **Большая пятёрка (Big Five)**: оценка по 5 факторам (открытость, добросовестность, экстраверсия, доброжелательность, нейротизм)
- **Локус контроля**: внутренний vs внешний (кто виноват в успехах/неудачах)
- **Уровень нарциссизма**: здоровый vs нездоровый, как проявляется
- **Толерантность к неопределённости**: как реагирует на хаос, неизвестность

### 3. Когнитивные паттерны

- **Стиль мышления**: аналитический vs синтетический vs интуитивный
- **Когнитивные искажения**: какие систематические ошибки мышления видны (черно-белое мышление, катастрофизация, сверхобобщение, и т.д.)
- **Уровень рефлексии**: насколько автор анализирует свои мысли, чувства, поступки
- **Метакогниция**: думает ли о том, как думает
- **Принятие решений**: импульсивность vs рациональность, на чём основывает решения

### 4. Мотивационная структура

- **Базовые потребности** (по Маслоу): какие уровни потребностей чаще всего озвучиваются
- **Внутренняя vs внешняя мотивация**: что движет автором
- **Ценностные конфликты**: где видны внутренние противоречия
- **Отношение к успеху и провалу**: как воспринимает, как описывает
- **Целеполагание**: долгосрочные vs краткосрочные, конкретные vs абстрактные

### 5. Самопрезентация и идентичность

- **Как автор видит себя**: самоидентификация, роли (предприниматель, отец, друг, лидер)
- **Как хочет быть увиденным**: какой образ транслирует
- **Разрыв между «я реальный» и «я публичный»**: где проскальзывает
- **Идентификация с группой**: кто «свои», кто «чужие»
- **Динамика идентичности**: как менялась самопрезентация по годам

### 6. Социальный интеллект

- **Обращение к аудитории**: как выстраивает отношения с читателями
- **Эмпатия**: проявляется ли в текстах, как именно
- **Границы**: что автор открывает, что скрывает
- **Отношение к критике**: как реагирует (если видно)
- **Юмор**: тип юмора (самоирония, ирония, сарказм, абсурд)

### 7. Защитные механизмы

- **Какие защиты использует**: юмор, рационализация, отрицание, проекция, сублимация
- **Реакция на стресс**: как проявляется в текстах
- **Уязвимые зоны**: о чём пишет уклончиво или с натянутым оптимизмом

## Формат результата

Вернуть **структурированный JSON**:

```json
{
  "skill": "author-analyze-psychology",
  "analysis": {
    "emotional_profile": { ... },
    "personality_type": { ... },
    "cognitive_patterns": { ... },
    "motivation": { ... },
    "self_presentation": { ... },
    "social_intelligence": { ... },
    "defense_mechanisms": { ... },
    "evolution_over_years": "как менялся автор по годам (от старых к новым постам)",
    "key_insights": ["инсайт 1", "инсайт 2", ...],
    "for_author_skill": "компактный абзац 300-500 слов: ключевые психологические черты, которые нужно воспроизвести в автор-скилле"
  }
}
```

**Критично:** Поле `for_author_skill` — это выжимка, которая пойдёт в финальный скилл. Она должна быть конкретной, actionable, с примерами из текстов автора.

## Делегирование саб-агентам

Этот скилл сам является задачей для саб-агента. Выполняется атомарно:
1. Прочитать файл `author_texts_for_analysis.json`
2. Провести анализ
3. Сохранить результат в `backend_python/data/author_analysis/psychology.json`
4. Вернуть краткое резюме (поле `for_author_skill`)
