# Примеры сообщений коммитов

## Формат

```
<тип>: <краткое описание на русском>

Изменения:
- <область>: описание (N файлов)
- ...

Файлов изменено: X | Новых: Y | Удалено: Z
```

## Примеры

### Одна область — один тип

```
feat: Массовое редактирование постов — новый модуль Bulk Edit

Изменения:
- Бэкенд: новый роутер bulk_edit.py, CRUD, схемы (5 файлов)
- Фронтенд: компонент BulkEditPanel, хук useBulkEdit (4 файла)
- Документация: docs/backend/features/bulk_edit/ (2 файла)

Файлов изменено: 0 | Новых: 11 | Удалено: 0
```

### Рефакторинг

```
refactor: Декомпозиция PostModal — вынос логики в хуки

Изменения:
- Фронтенд: features/posts/ — PostModal разбит на 6 хуков (12 файлов)
- Тесты: smoke-тест пройден, импорты ОК

Файлов изменено: 12 | Новых: 6 | Удалено: 0
```

### Баг-фикс

```
fix: Исправление публикации постов — неправильный формат даты

Изменения:
- Бэкенд: services/vk_api/publisher.py — формат ISO → Unix timestamp (1 файл)

Файлов изменено: 1 | Новых: 0 | Удалено: 0
```

### Смешанные (самый частый случай)

```
mix: Обновление CRUD, скиллы AI, UI автоматизаций

Изменения:
- Бэкенд: рефакторинг crud/ — единый стиль, новые методы (18 файлов)
- Фронтенд: UI автоматизации сторис — viewers_details (6 файлов)
- Скиллы AI: новые скиллы tester, documenter, git-push (15 файлов)
- Документация: обновлены docs/backend/ (4 файла)
- Инструкции: почищены .github/instructions/ (3 файла)

Файлов изменено: 34 | Новых: 15 | Удалено: 3
```

### Только документация

```
docs: Обновление документации бэкенда — automations, market

Изменения:
- Документация: docs/backend/features/automations/ (3 файла)
- Документация: docs/backend/features/market/ (2 файла)

Файлов изменено: 5 | Новых: 0 | Удалено: 0
```

### Только конфигурация

```
chore: Обновление скиллов AI и инструкций

Изменения:
- Скиллы: новые .agents/skills/tester, documenter (8 файлов)
- Инструкции: очищен ai_instr, удалены устаревшие (3 файла)

Файлов изменено: 3 | Новых: 8 | Удалено: 3
```

## Антипаттерны (НЕ делать так)

```
❌ "Update code"
❌ "fix"  
❌ "changes"
❌ "wip"
❌ "asdfgh"
❌ "Обновление проекта: сохранение текущего состояния"
```
