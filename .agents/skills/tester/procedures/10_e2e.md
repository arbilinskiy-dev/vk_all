# Процедура: E2E-тесты (Playwright)

**Триггер:** «e2e тест», «полный цикл», «playwright», «тест в браузере»
**Время:** 5-15 минут
**Приоритет:** 🟢 P2

## Предусловия

- Playwright установлен (`npm install -D @playwright/test && npx playwright install chromium`)
- Бэкенд запущен (localhost:8000) + фронтенд (localhost:5173)

6. [ ] При падении — скриншот + console.log → вердикт

## Типовые сценарии

1. **Авторизация → Дашборд** — открыть / → редирект → ввести токен → проверить дашборд
2. **Отправка сообщения** — авторизация → сообщения → отправить → проверить
3. **Создание поста** — авторизация → посты → создать → проверить список

## Важно

- E2E требуют запущенный бэкенд и фронтенд
- Использовать тестовый аккаунт (не прод)
- Установить только chromium (не все браузеры)
- Playwright MCP-сервер для прототипирования сценариев
