# Скроллбары

**Правило:** Любой контейнер с прокруткой (списки, сайдбары, таблицы, textarea) ОБЯЗАН использовать глобальный CSS-класс `.custom-scrollbar`.

## Правильно

```tsx
<div className="overflow-y-auto custom-scrollbar">
  {content}
</div>
```

## Неправильно

```tsx
// Отсутствие стилизации
<div className="overflow-y-scroll">

// Устаревший/сторонний класс
<div className="scrollbar-thin">
```

## Технические характеристики (определены в глобальном CSS)

- **Трек:** Прозрачный
- **Ползунок:** `#D1D5DB` (Tailwind `gray-300`)
- **Ползунок при наведении:** `#9CA3AF` (Tailwind `gray-400`)
- **Ширина/Высота:** 6px (тонкий)
- **Скругление:** 3px (полностью округлый)

## Где обязательно

- Сайдбар проектов
- Блок статистики историй
- История автоматизаций
- Любые горизонтальные списки с прокруткой (например, row с историями)
- Таблицы с вертикальной прокруткой
- Textarea с длинным текстом
